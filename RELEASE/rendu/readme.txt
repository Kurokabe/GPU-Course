*******************************************************
Rendu des TPs de Traitement Parallèle de Données
*******************************************************

Les TPs ont été réalisés par M. Paul Arzul, M. Charles-Lewis Jaggi
et M. Steven Jeanneret durant l'année 2018-2019 à la Haute École Arc de Neuchâtel.

Sont rendus les TPs suivants:

* DemoCube
* MandelBrot
* MontecarloInt
* MontecarloLong
* MontecarloMultiGPU
* RayTracing CM
* RayTracing GM
* RayTracing SM
* Reduction Tools
* Rippling
* Slice GM
* Slice SM

Les rendus possèdent la structure requise demandée dans le
fichier "Evaluation_TP_014.pdf".
